# Hostel Complaint Management System

## Project Title
**Hostel Complaint Management System (HCMS)**

## Objective
To design and develop a web-based system that allows hostel students to register, track, and manage complaints (maintenance, food, hygiene, security, etc.) digitally, while enabling hostel wardens/administrators to efficiently receive, assign, and resolve these complaints. The system aims to replace manual complaint registers with a centralized, transparent, and time-efficient digital platform, reducing resolution time and improving accountability.

## Module List

1. **User Authentication Module**
   - Student registration/login
   - Warden/Admin login
   - Role-based access control

2. **Student Complaint Module**
   - Register new complaint (category, description, room/block number, attachments)
   - View complaint status (Pending / In Progress / Resolved)
   - Edit or withdraw complaint before assignment
   - Complaint history

3. **Admin/Warden Dashboard Module**
   - View all complaints (filter by category, status, date, block)
   - Assign complaints to staff/technicians
   - Update complaint status
   - Add remarks/resolution notes

4. **Complaint Categorization Module**
   - Predefined categories: Electrical, Plumbing, Furniture, Internet, Food/Mess, Cleanliness, Security, Others
   - Priority levels (Low, Medium, High/Urgent)

5. **Notification Module**
   - Email/SMS/in-app alerts for complaint status updates
   - Reminders for unresolved complaints

6. **Staff/Technician Module**
   - View assigned complaints
   - Update work progress and mark completion

7. **Reports & Analytics Module**
   - Complaint statistics (category-wise, block-wise, resolution time)
   - Downloadable reports (PDF/Excel) for admin review

8. **Feedback & Rating Module**
   - Students rate resolution quality
   - Feedback comments after complaint closure

9. **Admin Management Module**
   - Manage student and staff accounts
   - Manage hostel blocks/rooms data
   - System configuration settings

10. **Database Module**
    - Store user data, complaint records, status logs, and feedback securely
